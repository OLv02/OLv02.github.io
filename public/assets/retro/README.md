# Retro Theme Assets

This directory contains assets for the retro 1990s-style theme.

## Current Assets (Placeholders)

All current assets are **SVG placeholders** that approximate the retro aesthetic.

### globe.svg
- **Current**: Static SVG globe with latitude/longitude lines
- **Recommended**: Replace with an actual animated `globe.gif` (spinning earth animation)
- **Authentic sources**:
  - Search for "spinning globe gif 90s"
  - Archive.org's GIF collection
  - Classic web graphics repositories

### bg-tile.svg
- **Current**: Simple gray checkered pattern (4x4px)
- **Recommended**: Replace with authentic tiled background texture
- **Suggestions**:
  - Classic marble texture
  - Dithered gradient patterns
  - Geometric repeating patterns from 90s web design

### runner.svg
- **Current**: Static SVG construction worker icon
- **Recommended**: Replace with animated `runner.gif` or `construction.gif`
- **Authentic sources**:
  - "Under construction" animated GIFs
  - Running construction worker animations
  - Classic "work in progress" clipart

## Adding Authentic Assets

To use real retro assets:

1. Find or create authentic 90s-style animated GIFs
2. Name them: `globe.gif`, `bg-tile.gif`, `runner.gif`
3. Replace the SVG files in this directory
4. Update CSS in `src/styles/theme-retro.css` if needed (file extensions)

The CSS is written to accept both `.svg` and `.gif` extensions.
